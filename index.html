We had a script that would start each one and do a simulation of disease outbreaks based on a preset configuration. There were 3 unique programs set up on the EC2 instances. In my distributed systems class we created a system that could calculate edit distances of a huge list string pairs. This was accomplished using Amazon EC2 instances and S3 buckets. There would be a 'manager' elected amongst the EC2 instances, then the rest would be assigned tasks, moving their tasks into the S3 buckets until all tasks were done and then communicated to the manager that the work was done. The manager could then safely shut everything down.
